👁️ GuideMe 🔉
Голосовой AI-помощник для незрячих и слабовидящих людей
========================================================
Версия: Alpha 1.0

О ПРОЕКТЕ
==========
AlphaVision - это инновационное веб-приложение, которое помогает 
незрячим и слабовидящим людям ориентироваться в пространстве. 
Используя камеру смартфона и искусственный интеллект, приложение 
в реальном времени распознаёт объекты и описывает их голосом.

"Технологии должны делать мир доступнее для каждого" - наш главный принцип.


ВОЗМОЖНОСТИ
============
✅ Распознавание объектов - определяет людей, машины, животных, предметы
✅ Голосовое описание - приятный женский голос описывает всё вокруг
✅ Приоритет людей - сообщает о появлении человека мгновенно
✅ Приоритет машин - мгновенно предупреждает о приближающемся транспорте
✅ Естественная речь - говорит живыми фразами
✅ Гибкие настройки - регулировка громкости, частоты подсказок, точности
✅ Адаптивный дизайн - для телефонов и планшетов
✅ Тёмная тема - для экономии батареи
✅ Визуальные рамки - для помощников (видят, что распознано)
✅ Журнал событий - подробный лог всех обнаружений


КАК ЭТО РАБОТАЕТ
=================
1. Камера смартфона снимает окружающее пространство
2. Искусственный интеллект анализирует изображение
3. Голосовой помощник описывает всё, что видит
4. Вы слышите описание и понимаете, что вокруг


БЫСТРЫЙ СТАРТ
==============
На телефоне:
1. Скачайте все файлы проекта
2. Откройте index.html в браузере (Chrome/Edge/Safari)
3. Разрешите доступ к камере
4. Нажмите "Запустить камеру"
5. Держите телефон так, чтобы камера смотрела вперёд

На компьютере:
1. Откройте папку с проектом
2. Запустите локальный сервер (например: npx http-server)
3. Откройте в браузере: http://localhost:8080

ЧАСТЫЕ ВОПРОСЫ
===============
В: Почему я ничего не слышу?
О: Проверьте:
   - Включён ли звук на устройстве
   - Разрешение на использование микрофона/звука
   - Настройки громкости в приложении

В: Не работает камера?
О: Убедитесь, что:
   - Вы разрешили доступ к камере
   - Камера не используется другим приложением
   - Вы используете современный браузер

В: Медленно работает?
О: Это нормально для первого запуска. Модель ИИ загружается в браузер.
